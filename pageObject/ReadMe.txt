Created a Feature for doing a simple google search for ducks and once the results are displayed retrieved all the hyperlinks in results and verified if they contains the text "duck"

Structure of Framework:
1. Implemented the Page Object in "pageObject\src\test\java\pageObject"
2. Implemented the tests of Page class in "pageObject\src\test\java\testCases"
3. Implemented step definitions for features in "pageObject\src\test\java\stepDefinitions"
4. Test Runner using junit is implemented in "pageObject\src\test\java\runner"

Features Implemented:
1. Selenium using java
2. Page Object Model framework is used
3. POM architecture is used
4. extentReports used
5. TestRunner class is configured in POM.xml for execution
